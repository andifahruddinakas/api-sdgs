API Sustainable Development Goals

Sistem API terpadu untuk mengakses data Sustainable Development Goals (SDGs) dari seluruh desa di Indonesia dengan antarmuka yang sederhana dan responsif.

Fitur Utama

Akses data SDGs dengan mudah dan cepat

Data Terstruktur

Data SDGs tersimpan dalam format JSON yang terstruktur rapi, memudahkan integrasi dengan aplikasi Anda.

  • Metadata lengkap
  • Gambar terintegrasi
  • Cache efisien

Update Otomatis

Generator berjalan otomatis setiap bulan untuk memastikan data selalu terkini tanpa intervensi manual.

  • Cron job Vercel
  • Error handling
  • Logging komprehensif

API Mudah Digunakan

Endpoint sederhana dan dokumentasi lengkap membuat integrasi menjadi cepat dan mudah.

  • REST API
  • No authentication
  • CORS enabled
